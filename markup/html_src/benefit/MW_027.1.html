LTE 데이터 충전

LTE 데이터 충전 (멤버십) 100M/300M/500M/1GB/2GB는 LTE 요금제를 사용하시는 고객님들을 위한 서비스입니다.
  • LTE 데이터 충전 서비스는 올레닷컴 사이트 (PC)에서 신청이 가능합니다.
  • 데이터 무제한 요금제, 청소년 요금제, 선불요금제 등은 신청이 불가합니다.
  • 충전된 데이터는 신청한 달의 다음달 말일까지 사용 가능하며, 미사용된 데이터는 소멸됩니다.
  • 데이터는 가장 먼저 등록한 순서대로 차감되며, 요금제 기본 제공량보다 먼저 차감됩니다.
  • 안심차단형 요금제 고객의 경우, 'LTE 데이터충전' 이용 중 요금제 변경 시 데이터 잔여량이 소멸될 수 있으니, 충전된 데이터 모두 사용 후 요금제 변경하시기 바랍니다.
